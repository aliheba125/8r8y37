# Hosted Files

## Structure:
- `binaries/sock64` - Original daemon binary from APK assets
- `binaries/bypass` - Anti-cheat bypass binary from res/raw
- `updates/latest.zip` - Update package containing:
  - `sock` - Updated daemon (replaces sock64)
  - `bypass` - Updated bypass binary
  - `loader/libpubgm.so` - Game hooking library (xhook-based)
